Nexus Sight — darmowa analiza graczy League of Legends (PL + AI)

Masz dość pustych statystyk typu KDA i win rate, które nic realnie nie mówią o Twojej grze?

Zbudowałem Nexus Sight — narzędzie, które łączy API Riot + analizę AI (Google Gemini) i pokazuje, jak naprawdę grasz — nie tylko czy wygrywasz.

---

🔍 Co dostajesz?

Profil gracza

- Wyszukiwanie po Riot ID (16 regionów)
- Ranga SoloQ / Flex + poziom konta
- Wykres LP (jak lecisz w górę… albo w dół 👀)
- Top 7 championów (mastery)

---

📊 Realna analiza (nie kosmetyka)

System liczy 12+ wskaźników, m.in.:

- OP Score (0–100) – Twoja faktyczna wydajność
- KDA Index – czy grasz czy feedujesz
- Kill Participation – czy jesteś częścią teamu czy NPC
- Vision Score – mapa vs ciemność
- CS/min – ekonomia gry
- Damage Share – ile naprawdę robisz
- Tilt Index – czy jedna przegrana rozwala Ci mental
- Playstyle AI – np. Assassin Carry / Tank Initiator
- Predicted Rank – gdzie powinieneś być według AI
